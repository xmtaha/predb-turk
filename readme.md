
Nedir?

PreDB Türk, BLURAY kategorisindeki Scene release'lerini takip etmek için geliştirilmiş, modern ve kullanıcı dostu bir web uygulamasıdır. PreDB.net API'sini kullanarak gerçek zamanlı release bilgilerine erişim sağlar.
Neden Yapıldı?

Film ve dizi severler için BLURAY release'lerini takip etmek bazen zor olabilir. Bu site tam da bu sorunu çözmek için geliştirildi:

    Hızlı Takip: Son çıkan BLURAY release'leri anında görün
    Kolay Arama: Belirli bir filmi veya diziyi hızlıca bulun
    NFO Görüntüleme: Her release'in NFO bilgilerine tek tıkla ulaşın
    Otomatik Güncelleme: Dakikada bir yeni release'ler otomatik yüklenir
    Offline Önbellek: Sürekli API isteği atmadan verileri kaydedin

Özellikler

    Son 50 BLURAY release'i anlık görüntüleme
    API üzerinden gerçek zamanlı arama yapabilme
    Her release için detaylı bilgiler (tarih, grup, durum)
    NFO görüntüleme ve indirme imkanı
    LocalStorage ile veri önbellekleme
    60 saniyede bir otomatik yenileme
    Geri sayım sayacı ile yenileme takibi
    Log sistemi ile işlem kayıtları
    Responsive tasarım - Mobil uyumlu
    Temiz ve minimalist arayüz

Teknik Detaylar

Bu proje tamamen Vanilla JavaScript ile geliştirilmiştir. Herhangi bir framework kullanılmamıştır. Modern web teknolojileri kullanılarak hızlı, güvenilir ve kullanıcı dostu bir deneyim sunulmuştur.

    API: PreDB.net Public API
    Frontend: HTML5, CSS3, JavaScript (ES6+)
    Depolama: LocalStorage API
    Tasarım: Custom CSS (Flexbox & Grid)
    Tema: Kırmızı-Beyaz minimalist

Sorumluluk Reddi

Bu uygulama yalnızca bilgi amaçlıdır. Hiçbir telif hakkıyla korunan içerik barındırmaz veya saklamaz. Tüm veriler PreDB.net'in halka açık API'sinden çekilmektedir. Scene release bilgileri sadece bilgilendirme amaçlı gösterilmektedir.
